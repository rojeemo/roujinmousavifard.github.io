Mobile Mapping Robot Algorithm

This project involves the design of an innovative algorithm to enable a mobile robot to navigate autonomously and generate detailed maps of indoor environments. The standout feature of this system is its reliance solely on an affordable ultrasonic sensor module to estimate obstacle distances, eliminating the need for expensive sensors like LiDAR. This cost-effective approach makes the robot highly accessible while maintaining accurate mapping and navigation capabilities.
